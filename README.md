# First Vue Project

Это мой первый проект на **Vue 3**.  
Проект сделан для практики современного Vue-стека: работа с состоянием, запросами к API, валидацией форм и базовой архитектурой приложения.

## Stack

Основные технологии, используемые в проекте:

- **Vue 3** — основной фреймворк
- **Vue Router** — маршрутизация
- **Pinia** — управление состоянием
- **TanStack Query (@tanstack/vue-query)** — работа с серверными запросами и кешированием
- **Zod** — валидация данных и форм
- **Tailwind CSS** — стилизация
- **Vuetify** — используется частично для UI компонентов
- **Lucide Icons / MDI** — иконки

## Dev Stack

Инструменты разработки:

- **Vite** — сборщик
- **TypeScript**
- **ESLint**
- **Oxlint**
- **Prettier**
- **vue-tsc** — type checking для Vue
